# Urban_Flow
UrbanFlow is a next-generation ride-hailing solution built on the MERN stack with a deeply integrated AI-powered chatbot. It enables real-time ride booking, driver tracking, fare estimation, and contextual automated support. Unlike conventional apps, UrbanFlow focuses on reducing driver distractions and improving customer satisfaction by automating repetitive interactions through an intelligent chatbot interface. 

## Tech Stack-
Frontend:	React.js (user & driver interfaces), Tailwind CSS (modern styling, responsive design)
Maps & Geolocation:	Google Maps API, Mapbox, Leaflet.js (routes, live driver tracking)
Real-Time Comm.:	Socket.io (client & server for bidirectional communication)
Backend:	Node.js + Express.js (RESTful APIs, server-side logic)
Database:	MongoDB + Mongoose (schema modeling for users, drivers, rides)
Authentication:	JWT (secure login/session management), bcrypt (password hashing)
Chatbot & AI:	Dialogflow CX / Botpress (NLP chatbot engine), Express.js Webhooks (intent handling), GPT API (contextual responses)
Cloud Functions:	Google Cloud Functions, AWS Lambda (scalable AI hosting)
Dev Tools:	Git & GitHub (version control), Postman (API testing & debugging)
